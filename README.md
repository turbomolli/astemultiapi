# astemultiapi
Multiapi project
## Getting started

### Clone project:
git clone https://github.com/turbomolli/astemultiapi.git

### to run backend:
cd astemultiapi/backend

npm install

npm run dev

### to run frontend:
npm install -g @angular/cli

cd astemultiapi/frontend/aste-app

npm install

ng serve
